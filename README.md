# 🌟 Portfolio Cá Nhân - Trần Thái Hoàng Phát

Chào mừng bạn đến với dự án **Portfolio Cá Nhân** của **Trần Thái Hoàng Phát** - học sinh trường **Tiểu học Trà An**. Đây là website được xây dựng để giới thiệu bản thân, quá trình học tập, các dự án đã thực hiện và thông tin liên hệ.

1. Clone repository:

```bash

```

2. Mở thư mục dự án:

```bash
cd your-repository
```

3. Mở file `index.html` bằng trình duyệt hoặc sử dụng Live Server trong VS Code.
